# Frequently Asked Questions

Common questions about OntExtract.

## General

### What is OntExtract?

OntExtract is a digital humanities platform for analyzing historical documents with period-aware processing and temporal evolution tracking.

### Do I need an API key to use OntExtract?

Core features work without an API key. LLM orchestration features require an Anthropic API key.
